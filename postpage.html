  Java basics -- String manipulation

  (String, StringBuffer, StringBuilder)

String

The String class inherits the object class, which describes strings. Java defines the String class to work with strings and provides a series of methods to manipulate strings, all wrapped in the java.lang package so you can use them without importing the package.

The String class uses the final modifier to indicate that the class cannot be overridden.


StringBuffer

Unlike String, the StringBuffer class operates on the object itself, while the String class automatically creates new objects.


The StringBuilder class

The StringBuilder class, introduced by Java SE5, is also a class that represents strings. It is similar to the StringBuffer class, but they differ in one important respect. StringBuilder, on the other hand, is not synchronous, meaning it is thread-unsafe and is typically used with a single thread.

The StringBuffer and StringBuilder apis are compatible, and StringBuilder is more efficient for replacing StringBuffer in single-threaded programs.
