Object & Class

Java is an object-oriented language. Everything in Java is associated with classes and objects, along with its attributes and methods. For example: in real life, a car is an object. The car has attributes, such as weight and color, and methods, such as drive and brake. Support the following basic concepts:

polymorphism; inheritance; encapsulation; abstract; class; object; The instance; methods; overloading


In this section we focus on the concepts of objects and classes.


Object: an object is an instance of a class (not a girlfriend) that has a state and behavior. For example, a dog is an object whose status is: color, name, breed; Behaviors include: tail wagging, barking, eating, etc.

Class: a class is a template that describes the behavior and state of a class of objects.

Objects


Now let's take a closer look at what an object is. If you look at the real world around you, you will find many objects, cars, dogs, people and so on. All of these objects have their own states and behaviors.


Take a dog for example. Its status is: name, breed, color, and behavior is: barking, wagging its tail, and running.


Compare real objects and software objects, and they are very similar.


Software objects also have states and behaviors. The state of a software object is an attribute, and the behavior is represented by a method.


In software development, methods manipulate changes in the internal state of objects, and objects call each other through methods.



Class


          A class can contain the following type variables:


          Local variables: variables defined in a method, constructor, or statement block are called local variables. Variables are declared and initialized in a method, and when the method ends, the variable is automatically destroyed.

          Member variables: a member variable is a variable defined outside the body of a method in a class. This variable is instantiated when the object is created. Member variables can be accessed by methods in a class, constructors, and class-specific blocks of statements.

          Class variables: class variables are also declared outside the body of the method in the class, but must be declared static.

A class can have multiple methods, in the example above: barking(), hungry(), and sleeping() are methods of the Dog class.



Constructor


Each class has a constructor. If a constructor is not explicitly defined for the class, the Java compiler will provide a default constructor for the class.

When creating an object, call at least one constructor. The constructor name must be the same as the class, and a class can have more than one constructor.



Create an object

         

          Objects are created from classes. In Java, the keyword new is used to create a new object. Creating an object requires the following three steps:

          Declaration: declares an object, including the object name and object type.

          Instantiation: use the keyword new to create an object.

          Initialization: when an object is created with new, the constructor is called to initialize the object.
